# Eclectica Restaurant

Учебный многостраничный сайт ресторана.  
Проект сделан в рамках обучения frontend-разработке.

Сайт состоит из нескольких страниц (главная, меню, контакты) и собирается с помощью Webpack.

## Стек

- HTML
- CSS
- JavaScript (ES-модули)
- CSS
- Webpack 5

## Установка и запуск

1. Установите зависимости: `npm install`
2. Запустите dev-сервер: `npm run start`
3. Соберите production-версию: `npm run build`
