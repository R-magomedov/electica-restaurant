# Eclectica Restaurant

Многостраничный сайт ресторана.



## О проекте
Проект сделан в рамках обучения frontend-разработке. Собирается через Webpack и оптимизируется для продакшена.
Готов к размещению на GitHub Pages (папка docs), ссылка:

https://r-magomedov.github.io/electica-restaurant/

## Стек

- HTML
- CSS
- JavaScript (ES-модули)
- CSS
- Webpack 5

## Запуск локально
npm install  
npm run start

## Сборка
npm run build
